# My dotfiles

dotfiles for my system

## Requirements

You need to have the following installed on your system

git,
stow,
fastfetch,
dunst,
waybar,
rofi
## Installation

First clone the repo

Then enter the directory of the cloned repo

```
$ git clone https://github.com/cunningsquid/dotfiles.git
$ cd dotfiles
```

Then use stow to create symlinks

```
$ stow .
```
